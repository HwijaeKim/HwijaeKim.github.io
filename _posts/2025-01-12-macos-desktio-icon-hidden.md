터미널 실행   
```bash
defaults write com.apple.finder CreateDesktop -bool FALSE; killall Finder
```


복귀
```bash
defaults write com.apple.finder CreateDesktop -bool TRUE; killall Finder
```
<br><br>

터미널 없이 손쉽게 가려주는 유틸리티도 있지만 유료
