# 🔑 핵심 기술 요약
- `Vanilla JavaScript`에 기반한 `Observer` 적극 활용으로 스크롤 시 `keyframe`애니메이션 구현
- CSS 가상클래스 사용으로 글꼴 크기, 색상 등을 공통으로 관리하여 코드 정리 및 유지보수에 이바지
- `svg`, `webp` 확장자 사용으로 페이지를 경량화
- Git을 이용한 GitHub 리포지토리를 Clone, 버전관리
