# 🔑 핵심 기술 요약
- CSS만을 사용하여 구현한 Windows 95~ME 스타일의 UI
- `Vanilla JavaScript`로 카운트, 남은 시간, 인게임 진행 등 프로그래밍 구현
- `localStorage`기능으로 사용자명을 로컬에 저장해 페이지가 전환되어도 호출할 수 있도록 프로그래밍
- CSS에서 마우스 커서 커스텀
- `JavaScript`에서 `setTimeout`, `setInterval` 지연실행, 초 단위 카운트 기능을 구현
